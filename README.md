**경기도 관광지/문화 시설 정보 앱을 제작함**

1. 주요 기능

    - 구글 지도에 관광지 및 시설 위치 표시 

    - 지도에서 마커를 클릭하면 상세 정보를 표시

    - 사용자가 원하는 지역, 카테고리(문화, 관광, 여가 등)를 선택하여 필터링 및 검색 기능

    - 사용자 위치 기반으로 가까운 시설 탐색

 

2. 사용한 데이터 (경기데이터드림) : https://data.gg.go.kr

    - 관광지 정보: 경기도의 주요 관광지 정보를 포함한 데이터셋

    - 문화 및 여가 시설: 도서관, 박물관, 문화회관, 체육 시설 등 다양한 문화 여가 시설 정보

 

3. 세부 요구 사항

    - 메인 화면 지도

    - 시설 데이터 로드 및 마커 표시

    - 시설 검색 및 필터링 기능

    - 사용자 위치 기반 가까운 시설 찾기

    - 즐겨찾기 기능

    - 기타 기능
